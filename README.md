# Safeeya Munawwar – Portfolio

A modern, responsive personal portfolio website built using React 19 and Vite.  
This portfolio showcases my projects, skills, certifications, and contact information with smooth animations and a clean UI design.

- React 19

- Tailwind CSS 4
- Framer Motion
- React Router DOM

- Fully responsive design

## ⚙️ Installation & Setup

### Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/safeeya-portfolio.git
```

### Navigate into the project folder:
```bash
cd safeeya-portfolio
```

### Install dependencies:
```bash
npm install
```

### Run locally:
```bash
npm run dev
```

### Build for production:
```bash
npm run build
```

### Preview production build:
```bash
npm run preview
```

---

## 🌍 Deployment

This project is deployed using Vercel.

To deploy:

1. Push project to GitHub
2. Connect repository to Vercel
3. Click Deploy

Vercel automatically detects Vite configuration.
